### API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| className | 组件类名 |   string   |   - |
| action | 后端地址 | string   | - | 
| defaultFileList | 上传文件列表 | UploadFile[] | [] |
| beforeUpload | 文件上传前的钩子函数 若返回 false 或者 Promise 则停止上传 | 	(file: File) => boolean | Promise| - |   
| onProgress | 文件上传时的钩子 | func | (percentage: number, file: UploadFile) => void | - |  
| onSuccess | 文件上传成功时的钩子 | func | (data: any, file: UploadFile) => void | - |
| onError | 	文件上传失败时的钩子 | func | (err: any, file: UploadFile) => void| - |  
| onChange | 文件状态改变时的钩子，上传成功或者失败时都会被调用 | func | (file: UploadFile) => void | - |  
| onRemove | 文件列表移除文件时的钩子 | func | 	(file: UploadFile) => void | - | 
| headers | 	设置上传的请求头部 | Obj | { [key: string]: any; } | - |  
| name | 上传的文件字段名 | 	string | (data: any, file: UploadFile) => void | - |
| data | 	上传时附带的额外参数 | Obj | 	{ [key: string]: any; }| - |  
| withCredentials | 	支持发送 cookie 凭证信息 | boolean | false | - |  
| accept | 可选参数, 接受上传的文件类型 | string| 	- | - |  
| multiple | 	是否支持多选文件 | boolean | false | - |  
| drag | 	是否支持拖拽上传| boolean | 	- | - |
