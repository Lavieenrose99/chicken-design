### 技术栈

- 开发语言和框架 TypeScript + React + Sass
- 构建工具 Gulp
- 静态站点 Docz
- 持续化部署 简单的采用git中的hooks里面的post-recieve
- 测试工具 Jest
- 代码约束 husky提交发布前验证 eslint提升代码规范性
- 评论功能待完善目前选用egg.js+mysql
