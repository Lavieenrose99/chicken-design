### API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| btnType | 按钮类型 | 'info'/'positive'/'negative'/'warning'非必填 | 'info' |
| className | 组件类名 |   string   |   - |
| size  | 大小 | 'sm' 'lg' | - | 
| disabled | 是否禁用 |boolean | false |
| href | 链接 | srting | - |
